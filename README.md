# Hausaufgabenheft

Im Hausaufagabenheft werden Hausaufgaben gespeichert. Immer neue Hausaufgaben können im Formular angelegt werden.

Die Einträge sollen in einer sqlite gespeichert werden.


## Menü

Das Menü hat folgende Einträge:

1.) Neue Hausaufgabe erstellen
2.) Hausaufgaben verwalten (löschen, bearbeiten)
3.) (Über-)fällige Hausaufgaben anzeigen
