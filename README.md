# Hausaufgabenheft

Das Online-Hausaufagabenheft hilft bei der Verwaltung von Hausaufgaben.
Einträge werden dauerhaft gespeichert und können angelegt, bearbeitet und wieder gelöscht werden.
Jeder neue Eintrag erzeugt eine Kalenderdatei (ICS), die der User herunterladen kann. Bei Klick auf die ICS-Datei wird am Vortag der Fälligkeit um 15 Uhr ein einstündiger Termin im persönlichen Kalender angelegt.

## Menü

Das Menü hat folgende Einträge:

1.) Neue Hausaufgabe erstellen
2.) Hausaufgaben verwalten (löschen, bearbeiten)
3.) Alle (über-)fälligen Hausaufgaben anzeigen

## Eigenschaften von Hausaufgaben

1.) Fach
2.) Thema
3.) Beschreibung / Aufgabe
4.) Fälligkeit

## Multiuser

Das Programm soll für mehrere User nutzbar sein. Dafür gibt sich jeder User einen Username. 
Mit Eingabe des Username bekommt der User seine persönlichen Einträge angezeigt und kann neue persönliche Einträge vornehmen.
Der Username soll am besten nach dem erneuten Aufruf der Webseite aus dem local storage geladen werden.

## Starten der App

Wenn ich F5 (oder die Playtaste) drücke soll die Webseite im neuen Tab öffnen.
